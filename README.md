*Sistema:* uma plataforma online para organização pessoal e recomendação de livros.

*Possíveis features:*
- cadastro de usuário
- cadastro dos livros

- lista de desejos
- recomendações de livros (com base nos gêneros dos livros já lidos)

*Possíveis tecnologias:*
- Front: React
- Back: Node
- Banco de dados: SQL
